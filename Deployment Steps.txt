1. Ensure that the setupvars.bat is initialized in command prompt/terminal https://docs.openvinotoolkit.org/latest/_docs_IE_DG_Samples_Overview.html

3. run the program saimese_ncs.py using python3

python saimese_ncs.py -d GPU

python saimese_ncs.py -d MYRIAD
